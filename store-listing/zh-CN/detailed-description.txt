Ladybird 是一款浏览器扩展，可让你无需编写代码，就能在网页上添加 UI 元素和由用户触发的自动化流程。

它能做什么

- 在网页中添加按钮、链接、工具提示和区域
- 创建包含点击、输入、等待、跳转、日志、分支、循环、确认弹窗的流程
- 使用 Hidden 规则隐藏不需要的页面元素
- 在 Overview 中按站点查看与删除已保存数据
- 在 Settings 中管理导入 / 导出和其他设置

核心功能

Elements / Flows / Hidden / Overview / Settings
- 在侧边栏中集中管理注入元素和自动化流程。
- 使用 Overview 按站点查看保存数据，并按需删除。

密码保管库（Password Vault）
- 可将流程输入步骤中重复使用的密码保存到本地加密的密码保管库。
- 密码字段会被策略保护，不能以明文保存到流程里。
- 运行流程时如果保管库处于锁定状态，Ladybird 可直接在页面内提示解锁，并从当前步骤继续执行。

导入 / 导出

- 可将保存的配置导出为 JSON 文件。
- 在确认保管库密码后，可选择将保管库数据一并导出。
- 支持导入配置数据，并可选恢复保管库数据。

多语言界面
- 支持英文 / 日文 / 简体中文界面。

典型使用场景

- 为常用网站添加快捷按钮
- 复用重复性的表单填写流程
- 创建带确认弹窗的半自动流程
- 隐藏不想看到的横幅或固定区域

隐私与安全边界

- 除非你主动导出，数据默认保存在本地 `chrome.storage.local`。
- 密码保管库数据在本地加密存储。
- Ladybird 不会把你的配置数据上传到遥测后端。
- 导出的文件需要你自行妥善保管。
- 如果忘记保管库密码，保管库内容无法找回，只能重置。

限制说明

- 网站 DOM 结构变化可能导致选择器失效。
- 某些使用严格 CSP 或高度动态 DOM 的网站可能影响注入稳定性。
- 跨域 iframe 自动化存在限制，不能保证所有场景都可用。
- 自动化行为依赖你的配置与页面状态，不保证对所有网站完全兼容。
