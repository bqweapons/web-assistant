Ladybird 是一款无需编写代码，就能为网页添加 UI 元素、隐藏规则和可复用自动化 Flow 的扩展。

它可以做什么

- 为页面添加按钮、链接、工具提示和区域
- 构建包含点击、输入、等待、assert、分支、循环、跳转、日志、数据源处理、运行时变量设置和弹窗确认的 Flow
- 使用手动 Hidden 规则隐藏不需要的页面元素
- 通过一个全局开关启用内置自动广告屏蔽
- 在 Overview 中查看保存的数据，并在 Settings 中管理导入 / 导出

核心功能

Elements / Flows / Hidden / Overview / Settings
- 通过侧边栏统一管理注入的 UI 元素和自动化 Flow。
- 在 Overview 中按站点查看保存的数据，并按需删除。

Flow 构建器
- 通过 `set-variable` step 创建运行时变量。
- 在后续 step 中使用 `{{var.name}}`、`{{loop.index}}`、`{{row.index}}` 等 token 复用值。
- 读取 CSV / TSV 数据源，并按每一行执行步骤。
- 在继续 Flow 之前显示带自定义消息的弹窗确认。

Hidden 规则
- 为当前站点添加手动隐藏规则。
- 内置广告屏蔽是全局功能，与手动规则列表分开。

密码保管库（Password Vault）
- 可将 Flow 输入步骤中复用的密码保存到本地加密的保管库中。
- 密码字段会受到策略保护，不允许以明文保存在 Flow 中。
- 即使侧边栏关闭，Ladybird 也可以在页面内提示解锁保管库，并从当前步骤继续执行。

导入 / 导出
- 可将保存的配置导出为 JSON。
- 导出时可在确认后附带 Password Vault 数据。
- 导入可恢复保存的配置和可选的保管库数据。

多语言 UI
- 支持英语 / 日语 / 简体中文。

典型使用场景

- 给常用网站添加快捷操作按钮
- 复用表单填写 Flow
- 运行带变量和弹窗确认的半自动流程
- 隐藏不想看到的横幅、重复 UI 或常见广告

隐私与安全

- 除非你主动导出，否则数据仅保存在本地 `chrome.storage.local`。
- Password Vault 数据在本地加密。
- Ladybird 不会把你的配置数据上传到遥测后端。
- 导出的文件需要你自行妥善保管。
- 如果忘记保管库密码，内容无法恢复，只能重置。

限制说明

- 某些网站频繁变更 DOM 结构时，选择器可能失效。
- 对 CSP 很严格或高度动态的页面，注入稳定性可能下降。
- 跨域 iframe 自动化存在限制，并非所有场景都可用。
- 自动化依赖你的配置和页面状态，无法保证对所有网站都完全兼容。
