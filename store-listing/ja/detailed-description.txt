Ladybird は、コードを書かずに Web ページへ UI 要素・非表示ルール・再利用可能な自動化フローを追加できる拡張機能です。

できること

- ボタン、リンク、ツールチップ、エリアをページに追加
- クリック、入力、待機、assert、分岐、ループ、移動、ログ、データソース処理、ランタイム変数設定、ポップアップ確認を含むフローを構築
- 手動の Hidden ルールで不要な要素を非表示
- 1 つのグローバル切り替えで組み込みの自動広告ブロックを有効化
- Overview で保存データを確認し、Settings でインポート / エクスポートを管理

主な機能

Elements / Flows / Hidden / Overview / Settings
- サイドパネルから挿入 UI 要素と自動化フローを管理できます。
- Overview でサイトごとの保存データを確認し、必要に応じて削除できます。

フロービルダー
- `set-variable` ステップでランタイム変数を作成できます。
- `{{var.name}}`、`{{loop.index}}`、`{{row.index}}` などのトークンで後続ステップから再利用できます。
- CSV / TSV のデータソースを読み込み、各行ごとにステップを実行できます。
- カスタムメッセージ付きのポップアップ確認を表示してからフローを続行できます。

Hidden ルール
- 現在のサイト向けに手動の非表示ルールを追加できます。
- 組み込みの広告ブロックはグローバル機能であり、手動ルール一覧とは分かれています。

パスワード保管庫（Password Vault）
- フローの入力ステップで再利用するパスワードを、ローカル暗号化された保管庫に保存できます。
- パスワード欄はポリシーにより平文保存から保護されます。
- サイドパネルが閉じていても、必要ならページ上で保管庫のロック解除を求め、そのまま現在のステップを続行できます。

インポート / エクスポート
- 保存した設定を JSON としてエクスポートできます。
- エクスポート時には確認後に Password Vault データを含めることもできます。
- インポートで保存設定と保管庫データを復元できます。

多言語 UI
- 英語 / 日本語 / 簡体字中国語に対応しています。

よくある用途

- よく使うサイトにショートカットボタンを追加する
- 定型入力フローを再利用する
- 変数やポップアップ確認付きの半自動フローを実行する
- 不要なバナー、繰り返し UI、よくある広告を隠す

プライバシーとセキュリティ

- データはエクスポートしない限り `chrome.storage.local` にローカル保存されます。
- Password Vault のデータはローカルで暗号化されます。
- Ladybird は設定データをテレメトリ用サーバーへ送信しません。
- エクスポートしたファイルの安全な保管はユーザー責任です。
- 保管庫パスワードを忘れた場合、内容の復元はできずリセットが必要です。

制限事項

- サイトの DOM 構造変更により、セレクタが壊れる場合があります。
- CSP が厳しいサイトや動的なページでは、挿入の安定性が下がる場合があります。
- クロスオリジン iframe の自動化には制限があり、すべてのケースで動作するとは限りません。
- 自動化は設定内容とページ状態に依存するため、サイト互換性は保証されません。
