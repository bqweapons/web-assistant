Ladybird は、コードを書かずに Web ページへ UI 要素とユーザー操作型の自動フローを追加できる拡張機能です。

できること

- ボタン、リンク、ツールチップ、エリアをページに追加
- クリック、入力、待機、移動、ログ、分岐、ループ、確認ポップアップを含むフローを作成
- Hidden ルールで不要な要素を非表示
- Overview でサイトごとの保存データを確認・削除
- Settings でインポート / エクスポートや各種設定を管理

主な機能

Elements / Flows / Hidden / Overview / Settings
- サイドパネルから注入要素と自動フローを管理できます。
- Overview で保存済みデータをサイト単位で確認し、必要に応じて削除できます。

パスワード保管庫（Password Vault）
- フローの入力ステップで再利用するパスワードを、ローカル暗号化された保管庫に保存できます。
- パスワード欄は平文のままフローに保存できないよう保護されています。
- 実行中に保管庫がロックされている場合は、ページ上で解除を促し、現在のステップから続行できます。

インポート / エクスポート
- 保存済み設定を JSON としてエクスポートできます。
- 保管庫パスワードの確認後、必要に応じて保管庫データを含めてエクスポートできます。
- 設定データと（任意で）保管庫データをインポートして復元できます。

多言語 UI
- 英語 / 日本語 / 簡体字中国語に対応しています。

よくある使い方

- よく使うサイトにショートカットボタンを追加する
- 繰り返し入力するフォーム操作をフロー化する
- 確認ポップアップ付きの半自動フローを作る
- 見たくないバナーや領域を非表示にする

プライバシーとセキュリティ

- データは、エクスポートしない限り `chrome.storage.local` にローカル保存されます。
- パスワード保管庫のデータはローカルで暗号化されます。
- Ladybird は設定データをテレメトリ用サーバーへ送信しません。
- エクスポートしたファイルの管理はユーザーの責任です。
- 保管庫パスワードを忘れた場合、内容は復元できず、リセットが必要です。

制限事項

- サイト側の DOM 構造変更により、セレクタが動作しなくなる場合があります。
- 厳しい CSP や動的 DOM の多いサイトでは、注入の安定性が下がることがあります。
- クロスオリジン iframe の自動化には制限があり、すべてのケースで動作するとは限りません。
- 自動化はユーザー設定とページ状態に依存するため、すべてのサイトでの完全動作は保証されません。
