This release expands flow authoring and simplifies built-in ad blocking.

- Added runtime variables with `set-variable`, including reusable `{{var.name}}` tokens across later steps.
- Added `{{loop.index}}` and `{{row.index}}` tokens for loop and data-source steps.
- `set-variable` can now read values from page elements by selector.
- Improved the step editor with variable shortcuts in more fields and more consistent mode-switch tabs.
- Automatic ad blocking is now a built-in global feature instead of per-site stored rules.
- Hidden Rules UI now separates manual site rules from the global automatic ad-blocking toggle.

Also includes UI, i18n, and flow-runner polish across recent screens.
