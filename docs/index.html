overview

サイドパネルレイアウト紹介

左ペインで要素追加と検索、中央でページ別の一覧、右ペインで設定・バックアップを操作する三分割レイアウトです。

トップビューから配置編集や新規作成に即アクセスでき、保存済み要素の検索や一括削除も可能です。ページ単位でエクスポート／インポートすれば、他デバイスやチームと共有できます。

  • ホーム: 追加・編集・テストをまとめたトップ。編集モードで配置済み要素をハイライトし、ドラッグやクリックで即編集。
  • 要素一覧: ページ別の保存要素を一覧化。検索・フィルター・一括削除、さらにページごとのバックアップ/復元が可能。
  • 設定: 言語やバックアップ、エクスポート/インポートを管理。共有用ストアリンクのコピーもここから。

この3タブを押さえたら、次は要素の追加に進みます。

add button

ボタンの追加

最も基本的な要素。ページ上に仮配置したあと、右ペインでテキストや色、リンク先を整えます。

  1. サイドパネル左で「要素タイプ」に「ボタン」を選択。
  2. 「要素を追加」をクリックするとページ上に仮配置されます。
  3. 右ペインでテキスト・色・リンク先を設定し、必要ならドラッグで位置を調整します。
action flow

Action Flowの追加

1つのボタンにクリック、入力、スクロールなど複数アクションを順番に紐づけ、フローとして自動実行できます。

  1. 追加したボタンを選択し、Action Flow編集を開きます。
  2. 「ステップを追加」からクリック・入力・スクロールなどを順番に追加。
  3. 並び替えや削除でフローを調整し、保存すると次のクリックから自動実行されます。
area

Areaの追加

任意の矩形エリアを操作対象として保存。UI変化に合わせて複数エリアを使い分けできます。

  1. 編集モードに入り、「Area」を選択。
  2. ページ上をドラッグしてエリアを囲み、保存します。
  3. 保存したエリアはActionのターゲットとして何度でも再利用できます。
fine tune

位置調整

配置後はドラッグで再調整。右ペインに座標がリアルタイム表示されるので、ピクセル単位の微調整が簡単です。

  1. 編集モードで対象の要素をドラッグして移動。
  2. 右ペインの座標を見ながら微調整。
  3. クリックで確定。別ページに移動しても位置は保持されます。
