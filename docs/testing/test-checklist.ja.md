# テストチェックリスト（docs テストページ）

## スモーク（5〜10分）

- [ ] `docs/test-pages/basic/test-page.html` を開く
- [ ] ボタン要素を1つ作成し、表示されることを確認
- [ ] popup フローを1つ実行し、`OK` で続行されることを確認
- [ ] hidden ルールを1つ作成し、ページ要素が非表示になることを確認
- [ ] Overview で保存済みサイトデータが見えることを確認

## Vault フロー

- [ ] フローの password 入力欄を Password Vault に紐付ける
- [ ] サイドパネルを閉じた状態でフローを実行
- [ ] ページ内の保管庫解錠プロンプトが表示されることを確認
- [ ] 間違った保管庫パスワード入力時に再試行できることを確認
- [ ] 正しいパスワードで現在ステップから再開されることを確認
- [ ] 解錠プロンプトをキャンセルし、ユーザー向けエラーで停止することを確認
- [ ] 保管庫をリセットし、既存紐付けが無効状態として表示されることを確認
- [ ] 無効な紐付けを再紐付けし、再実行成功を確認

## Data Source A/B

- [ ] `docs/test-pages/data-source/datasource-form-a.html` を開く
- [ ] 複数フィールド（`text`, `email`, `date`, `number`）をフローで入力
- [ ] Screen B へ送信
- [ ] 結果テーブルに値が表示されることを確認
- [ ] 戻りリンクで Screen A に戻る

## インポート / エクスポート（保管庫データ任意）

- [ ] 既定ルート（保管庫パスワードなし）でエクスポート
- [ ] 保管庫パスワード付きエクスポート（確認ダイアログ + 保管庫パスワード入力）
- [ ] JSON をインポートし、elements/flows が復元されることを確認
- [ ] 保管庫データ付きの場合、保管庫作成/解錠後に secrets が復元されることを確認

## i18n 確認

- [ ] サイドパネル言語を `en`, `ja`, `zh_CN` に切り替える
- [ ] tabs/header 表示が正しく切り替わる（英語残留なし）
- [ ] runner/vault 関連メッセージがローカライズされている
