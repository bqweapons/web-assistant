# Ladybird 隐私政策

最后更新：2026-02-26

## 概要

- 我们不会收集、出售或共享个人数据。
- 配置数据存储在本地 `chrome.storage.local` 中。
- Password Vault 数据仅在本地加密保存，只有解锁保管库时才会解密。
- 扩展不会将你的数据上传到 Ladybird 服务器（没有遥测后端）。

## 详细说明

### 1. 数据收集
Ladybird 不收集分析、遥测或使用行为数据。

### 2. 本地存储
- 扩展配置和已保存的元素/流程保存在本地 `chrome.storage.local` 中。
- Password Vault 数据以加密形式保存在本地。

### 3. 导出 / 导入
- 当你选择导出数据时，JSON 文件会在你的设备本地生成。
- 导出时可在输入保管库密码确认后选择包含 Password Vault 数据。
- 导出文件的安全保管由用户自行负责。

### 4. 权限用途
权限仅用于在页面中渲染用户定义的 UI、管理侧边栏，以及执行由用户触发的自动化流程。

### 5. 第三方
Ladybird 不会向第三方出售或共享你的数据。

### 6. 安全说明
- 扩展仅运行随安装包提供的代码（不执行远程代码）。
- Password Vault 解锁提示与流程日志会尽量避免保存或记录保管库密码明文。
- 如果忘记保管库密码，保管库内容无法恢复，只能重置。

### 7. 联系方式
如有隐私相关问题，请在本仓库提交 Issue。
